DVVActionSheetView
==================
自定义的ActionSheetView，解决了UIAlertController在iPad和iPhone上使用方式不一致的问题。

使用方式
-------
在`Podfile`里添加此行内容：
```
pod 'DVVActionSheetView', :git => 'https://github.com/devdawei/DVVActionSheetView.git', :tag => 'v1.0.1'
```

然后在`Terminal`中`cd`到工程目录，执行如下命令：
```
pod install
```
